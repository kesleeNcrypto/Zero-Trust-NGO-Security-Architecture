# Resume Bullets — Zero Trust NGO Security Project

## Security Architecture

- Designed a Zero Trust security architecture for a multi-country NGO operating in high-risk environments, securing distributed users, devices, cloud services, and sensitive data.

- Developed identity-first access controls using MFA, RBAC, PAM, conditional access, and Zero Trust Network Access (ZTNA) principles.

---

## Threat Modeling

- Built an NGO-specific threat model covering phishing, insider threats, partner compromise, device seizure, nation-state surveillance, and cloud abuse scenarios.

- Mapped adversary behaviors to MITRE ATT&CK techniques to improve threat visibility and detection strategy.

---

## Security Controls

- Defined layered controls across identity, endpoint, network, application, and data security domains aligned to Zero Trust principles.

- Designed remote wipe, device posture validation, encryption, segmentation, and DLP controls for high-risk field operations.

---

## Detection Engineering

- Created SOC detection use cases for suspicious logins, privilege escalation, mass downloads, logging disablement, and partner access abuse.

- Aligned detections to common SIEM workflows using Wazuh / Splunk style monitoring logic.

---

## Governance & Compliance

- Mapped controls to NIST CSF, NIST SP 800-207, ISO 27001, and CIS Controls v8 to support audit and donor assurance readiness.

- Produced executive-level security briefings communicating risk posture, control maturity, and strategic roadmap.

---

## Fraud Risk Management

- Designed fraud threat scenarios covering payroll fraud, vendor abuse, grant diversion, beneficiary identity fraud, and insider collusion.
